# AGENTS.md

## Project Name

Direkt IT Pomoć Zagreb

## Project Type

Croatian landing page for a small local IT help / computer repair / computer troubleshooting service in Zagreb.

## Main Goal

Build a trustworthy, human, SEO-friendly, responsive landing page for users who need help with:

- slow Windows computers
- laptop and desktop PC troubleshooting
- Windows optimization
- basic software setup
- backup and data transfer
- basic malware/suspicious software cleanup
- beginner-friendly IT help
- simple home-office / home-user technical problems

The website should help a user quickly understand:

1. what problem this service solves
2. what services are offered
3. what the approximate prices are
4. who is behind the service
5. how to contact the owner
6. what is not offered

## Owner / Brand Context

The owner is Mate / Vrachox.

He is an early-career IT technician / help desk / sysadmin-oriented person building practical IT skills and a small local service.

He should be positioned as:

- honest
- direct
- practical
- beginner-friendly
- affordable but not suspiciously cheap
- transparent about what he can and cannot fix

Important brand promise:

> If I know how to solve it, I will solve it. If the issue is outside my current level or requires a specialized repair shop, I will say so clearly.

## Target Audience

Primary audience:

- home users in Zagreb
- people with slow laptops or desktop PCs
- students
- older users
- people who are not technical
- small business owners / freelancers with simple IT problems
- users who want simple explanations instead of technical jargon

Typical user problems:

- "My computer is slow."
- "Windows is acting weird."
- "I need to save my files before reinstalling Windows."
- "I do not know what I can safely delete."
- "I need basic programs installed."
- "I need help with email, browser, printer, Wi-Fi, or settings."
- "I want someone honest who will not overcharge me."

## Tone of Voice

Use Croatian as the primary language.

The tone should be:

- human
- honest
- calm
- direct
- clear
- local
- practical
- respectful
- non-corporate

Avoid:

- fake enterprise language
- generic SaaS startup copy
- exaggerated claims
- empty buzzwords
- aggressive sales language
- pretending to be a large company
- overpromising

Good tone examples:

- "Objasnim jednostavno što se događa."
- "Cijena se dogovara prije rada."
- "Ako problem nije za moj nivo, reći ću odmah."
- "Cilj je riješiti problem, ne komplicirati."

Bad tone examples:

- "Najbolji servis u Zagrebu."
- "Garantirano rješavamo sve kvarove."
- "Premium enterprise IT solution."
- "Revolucionarna računalna transformacija."

## Services Offered

The website may present these services:

### Windows Help

- Windows optimization
- startup app cleanup
- basic update checks
- basic driver checks
- browser cleanup
- basic system organization
- simple troubleshooting

### Slow Computer Help

- identify common slowdown causes
- remove unnecessary startup items
- uninstall unnecessary software
- basic performance improvement
- practical advice for future maintenance

### Backup and Data Transfer

- backup important user files
- transfer documents/photos/basic files
- prepare files before reinstalling Windows
- explain what is being backed up

### Software Setup

- install basic programs
- browser setup
- email setup help
- PDF reader / office alternatives / utilities
- basic configuration

### Basic Security Check

- basic suspicious software review
- remove obviously unwanted apps
- browser extension review
- basic malware scan guidance
- practical security advice

### Beginner-Friendly IT Help

- explain how to use basic computer features
- help older users
- explain simple Windows usage
- help with everyday technical confusion

### Purchase / Upgrade Advice

- basic laptop/PC buying advice
- SSD/RAM upgrade advice at a high level
- explain whether repair or replacement makes more sense

## Services Not Offered

Do not claim or imply that the owner offers:

- advanced data recovery from physically damaged drives
- motherboard repair
- soldering
- liquid damage repair
- complex electronics repair
- guaranteed virus removal in every case
- illegal/pirated Windows or Office activation
- advanced corporate IT support
- cybersecurity penetration testing
- enterprise networking
- guaranteed repair of every problem
- work that voids warranty without warning

If mentioned, phrase honestly:

> Za fizički oštećene diskove, matične ploče, lemljenje, oštećenja od tekućine i slične napredne kvarove preporučujem specijalizirani servis.

## Pricing Direction

The website should show transparent approximate pricing.

Recommended package structure:

### Mini pomoć — od 20 €

For smaller issues such as email, browser, printer, basic settings, software help, or quick troubleshooting.

### Windows Refresh — oko 45 €

For slow Windows computers: basic system review, startup apps, unnecessary programs, updates, drivers, browser cleanup, and practical maintenance advice.

### Backup + sređivanje — od 55 €

For users who want important files backed up before deeper cleanup or reinstall.

### Reinstalacija Windowsa — od 45 €

Only if the owner wants to offer this. Must clearly separate:

- without backup
- with backup

### Kućna IT pomoć — po dogovoru

For beginner-friendly help, explanations, setup, and local assistance.

Rules:

- Prices are approximate.
- Do not pretend prices are fixed for every situation.
- Make it clear that the price is agreed before work starts.
- Keep pricing affordable but professional.
- Do not race to the bottom.

Good pricing copy:

> Cijena se dogovara prije rada. Ako se tijekom pregleda pokaže da je problem veći nego što izgleda, prvo objasnim što sam našao i koliko bi dalje koštalo.

## Website Sections

The homepage should include these sections in this order:

1. Hero
2. Common Problems
3. Services
4. Pricing Packages
5. How It Works
6. About Me
7. What I Do Not Do
8. FAQ
9. Contact CTA

### 1. Hero

Goal:
Immediately show that this is local, practical IT help in Zagreb.

Hero should include:

- strong headline
- short human subheadline
- primary CTA
- secondary CTA
- 2-4 trust notes

Suggested copy direction:

Headline:
"Računalo sporo? Windows zeza? Trebate normalnu IT pomoć u Zagrebu?"

Subheadline:
"Pomažem s osnovnim i srednje zahtjevnim problemima na računalima i laptopima: spori Windows, backup, programi, postavke i svakodnevne tehničke poteškoće — jasno, pošteno i bez kompliciranja."

CTA:
"Pošalji problem"

Secondary CTA:
"Pogledaj cijene"

Trust notes:
- "Cijena dogovorena unaprijed"
- "Bez tehničkog preseravanja"
- "Ako ne znam, kažem odmah"
- "Zagreb i okolica"

### 2. Common Problems

Show problems in simple cards:

- Računalo se sporo pali
- Windows izbacuje greške
- Laptop je zatrpan programima
- Trebate backup prije reinstalacije
- Ne znate što smijete obrisati
- Trebate pomoć s mailom, browserom ili postavkama

### 3. Services

Show clear service cards:

- Windows optimizacija
- Backup i prijenos podataka
- Instalacija programa
- Osnovna sigurnosna provjera
- Pomoć s internetom, mailom i preglednikom
- Savjetovanje prije kupnje

### 4. Pricing Packages

Use clear cards with approximate pricing.

Cards should be easy to scan on mobile.

Each card should include:

- package name
- price
- who it is for
- included items
- short note

### 5. How It Works

Simple step-by-step process:

1. Pošaljete problem
2. Dobijete kratku procjenu
3. Dogovorimo cijenu
4. Rješavam problem
5. Dobijete objašnjenje i savjet

### 6. About Me

Explain honestly:

- Mate / Vrachox
- Zagreb
- early-career IT technician / help desk / sysadmin direction
- practical IT learning and service
- honest approach
- focus on helping normal users

Avoid sounding like a large company.

### 7. What I Do Not Do

This section increases trust.

Mention:

- no pirated software
- no advanced physical data recovery
- no motherboard/liquid damage repair
- no fake guarantees
- no pretending

### 8. FAQ

Include FAQ items:

- Radite li dolazak na lokaciju?
- Mogu li prvo poslati opis problema?
- Što ako se problem ne može riješiti?
- Radite li backup podataka?
- Radite li piratski Windows/Office?
- Koliko traje sređivanje računala?
- Je li cijena unaprijed poznata?
- Možete li pomoći starijim osobama?

### 9. Contact CTA

Final section should be direct.

Include:

- phone placeholder
- WhatsApp placeholder
- email placeholder
- Zagreb location
- short form only if functional
- if no form backend exists, use mailto/WhatsApp links instead

## SEO Requirements

Use natural local SEO.

Primary SEO phrases:

- servis računala Zagreb
- servis laptopa Zagreb
- IT pomoć Zagreb
- pomoć oko računala Zagreb
- instalacija Windowsa Zagreb
- optimizacija Windowsa
- backup podataka Zagreb
- sporo računalo pomoć
- računalna pomoć Zagreb
- tehnička pomoć za računalo

Rules:

- Do not keyword-stuff.
- Use keywords naturally in headings, copy, metadata, FAQ, and alt text.
- Croatian language first.
- Make the page useful for real humans first.

## Technical SEO

Implement when relevant:

- strong metadata title and description
- Open Graph metadata
- sitemap.ts
- robots.ts
- LocalBusiness structured data if enough real business/contact data is available
- clear heading hierarchy
- semantic HTML
- accessible links and buttons
- good mobile layout
- optimized images

Use official Next.js App Router conventions for metadata, sitemap, and robots files.

## Tech Stack

Use:

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Vercel deployment

Optional:

- shadcn/ui for accessible UI primitives if useful
- Framer Motion only for subtle animations
- lucide-react for simple icons if already installed or intentionally added

Avoid:

- unnecessary dependencies
- heavy animation libraries for no reason
- complex backend
- fake working contact forms
- database/auth/payment unless explicitly requested later
